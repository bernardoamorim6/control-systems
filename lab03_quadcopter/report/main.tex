\section{Equilibrium point}

The state of the system is given by the vector \(\mathbf{x} = [x, y, z, v_x, v_y, v_z, \phi, \theta, \psi, \omega_x, \omega_y, \omega_z]^T\)
The control is defined as \(\mathbf{u} = [v_1^2, v_2^2, v_2^3, v^2_4]^T\)

\begin{align}
    \dot{x} &= v_x = 0\\
    \dot{y} &= v_y = 0\\
    \dot{z} &= v_z = 0\\
    \dot{v}_x &= 0\\
    \dot{v}_y &= 0\\
    \dot{v}_z &= -g + \frac{k\,c_m}{m} v_1^2 + \frac{k\,c_m}{m} v_2^2 + \frac{k\,c_m}{m} v_3^2 + \frac{k\,c_m}{m} v_4^2\\ 
    \dot{\phi} &= \omega_x = 0\\
    \dot{\theta} &= \omega_y = 0\\
    \dot{\psi} &= \omega_z = 0\\
    \dot{\omega}_x &= \frac{L\,k\,c_m}{I_{xx}} v_1^2 - \frac{L\,k\,c_m}{I_{xx}} v_3^2\\
    \dot{\omega}_y &= \frac{L\,k\,c_m}{I_{yy}} v_2^2 - \frac{L\,k\,c_m}{I_{yy}} v_4^2\\
    \dot{\omega}_z &= \frac{b\,c_m}{I_{zz}} v_1^2 - \frac{b\,c_m}{I_{zz}} v_2^2 + \frac{b\,c_m}{I_{zz}} v_3^2 - \frac{b\,c_m}{I_{zz}} v_4^2\\
\end{align}
